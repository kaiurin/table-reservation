# REST API on Express

REST API service to reserve the table.
You can POST, GET, PUT, DELETE
Date format: 'DD/MM/YYYY HH:mm'.

# Getting started

**1. Install modules**
```bash
$ npm install
```
**2. Configure your database**

- Create database 'restaurant';

- Execute _restaurant.sql_ file;

**3. Run your server**
```bash
node app.js
```
